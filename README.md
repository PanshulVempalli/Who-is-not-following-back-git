
# Who-is-not-following-back-git 

A simple and efficient tool designed to help GitHub users manage their community by identifying accounts that haven't followed them back.

## 🚀 Features
- **Fast Comparisons:** Quickly compares your 'Following' list against your 'Followers' list.
- **Clean Output:** Generates a clear list of usernames for easy review.
- **GitHub API Integration:** Uses official endpoints for accurate data.
